# LeetCode Problem Solving Journey

Most people scroll through problems.

Very few sit down and train their mind to solve them.

This repository documents my journey of solving algorithmic challenges from LeetCode — not just to write code, but to sharpen problem-solving, pattern recognition, and computational thinking.

Each solution represents a small step toward mastering algorithms and becoming a better engineer.

Consistency beats intensity.
One problem at a time.

The purpose of this repository is to:

• Build strong problem-solving skills  
• Master data structures and algorithms  
• Recognize common coding interview patterns  
• Maintain a consistent daily coding habit
